# mem-mcp web

Next.js 15 web interface for mem-mcp.

## Quick start

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:8081`.

## Available scripts

- `npm run dev` — Development server (port 8081)
- `npm run build` — Production build
- `npm run start` — Start production server (port 8081)
- `npm run lint` — Run ESLint

## Architecture

- **Next.js 15** App Router
- **TypeScript** strict mode
- **Tailwind CSS** for styling
- **React 18** for UI components

## Components

shadcn-style components are scaffolded in `components/`. Install specific ones as needed via later PRs.
